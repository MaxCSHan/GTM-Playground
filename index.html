Google Tag Manager とは？

Google Tag Manager は、Google による無料のタグ管理ツールです。このツールを使用すると、ウェブサイトに手動でコードを追加する必要なく、タグを簡単にデプロイおよび管理できます。タグは、ウェブサイトのさまざまなユーザーアクションをトラッキングするコードスニペットです。

GTM コンテナコードのインストール

最初のコードブロックは、ウェブページの <head> 開始タグの直後、または <head> のできる限り最初の方に配置することをおすすめします。 
                
      <!-- Google Tag Manager -->
      <script>
          (function (w, d, s, l, i) {
            w[l] = w[l] || [];
            w[l].push({ "gtm.start": new Date().getTime(), event: "gtm.js" });
            var f = d.getElementsByTagName(s)[0],
              j = d.createElement(s),
              dl = l != "dataLayer" ? "&l=" + l : "";
            j.async = true;
            j.src = "https://www.googletagmanager.com/gtm.js?id=" + i + dl;
            f.parentNode.insertBefore(j, f);
          })(window, document, "script", "dataLayer", "GTM-T9JL5SRN");
        </script>
        <!-- End Google Tag Manager -->
次に、ウェブサイトのすべてのページで、<body> 開始要素の直後に次のコードを追加します。 
                
    <!-- Google Tag Manager (noscript) -->
    <noscript>
      <iframe
        src="https://www.googletagmanager.com/ns.html?id=GTM-T9JL5SRN"
        height="0"
        width="0"
        style="display: none; visibility: hidden;"
      ></iframe>
    </noscript>
    <!-- End Google Tag Manager (noscript) -->
上記のインストール方法はウェブサイト開発者の協力が必要です。​
